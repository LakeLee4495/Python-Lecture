4.기본문법
4.1 세미콜론
	; 를 이용하여 여러줄을 한줄로 작성 가능
	예시) 	print('Hello')
		print('1234')       ->  print('Hello'); print('1234')

4.2 주석
	# 를 이용해서 사용
	alt + shift + a 단축키 이용하여 여러줄의 문자열 주석처리 가능

4.3 들여쓰기
	줄맞추기
	파이썬은 줄맞추기 재대로 안하면 들여쓰기 오류 뜸
	예시)	a = 5
		if a == 10;
		print('10입니다')
		else:
		print('10이 아닙니다')      <- 이렇게쓰면 오류

4.4 코드 블록
	if a == 10;
		print('10')
	쉬프트 + 탭 키 누르면 백탭, 그냥 탭키만 누르면 줄 정렬
